# 💧 TukarMinyak.Id - Sistem Pengelolaan Limbah

Aplikasi manajemen pengumpulan minyak jelantah berbasis web yang dirancang untuk membantu masyarakat mengelola limbah rumah tangga menjadi poin yang dapat ditukarkan dengan berbagai kebutuhan pokok (sembako).

## ✨ Fitur Utama
- **Dashboard Interaktif**: Visualisasi statistik minyak terkumpul dan estimasi saldo kas.
- **Sistem Poin**: Kalkulasi poin otomatis (1 Liter = 1 Poin) dengan integrasi notifikasi SweetAlert2.
- **Katalog Penukaran**: Interface katalog sembako modern dengan fitur konfirmasi transaksi yang aman.
- **Riwayat Transaksi**: Log aktivitas lengkap untuk memantau setoran dan penukaran barang.

## 🛠️ Tech Stack
- **Framework:** Laravel 11
- **Frontend:** Tailwind CSS
- **Interactions:** SweetAlert2
- **Fonts:** Inter
